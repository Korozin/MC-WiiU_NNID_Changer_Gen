# MC-WiiU_NNID_Changer_Gen
Run the .py file and when it says NNID: type the desired name.

After you type the NNID you'd like put in your IP when it asks for it

Example of usage

python NNID-Change.py

NNID: Name Test

Wii U IP: 192.168.1.97

Success!
